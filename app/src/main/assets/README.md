# RealKick 3D Assets Folder

Place your `.glb` 3D model files in this folder:
- `porteria.glb` (Football Goal)
- `balon.glb` (Football Ball)
- `copa_del_mundo.glb` (World Cup Trophy)

Ensure they are named exactly as shown above (case-sensitive, lowercase) and are in standard binary glTF (`.glb`) format.
